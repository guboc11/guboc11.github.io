Go 다운로드 & 설치

설치 링크

터미널에

go verion

이라고 작성하고 실행했을 때 설치된 Go 언어 버전이 출력되면 올바르게 설치 완료된 것이다.

파일 생성

우리는 Go Package를 생성하려는 것이 아니라 그냥 빠르게 실행해보는 목적이니 go mod init 명령어는 따로 실행하지 않아도 된다.

터미널, 메모장, VS Code 등 어떤 방법으로든 일단 main.go라는 파일을 생성해보자.

터미널에서는

touch main.go

명령어로 생성할 수 있다.

그리고 Vim, 메모장, VS Code 등 자신이 원하는 에디터로 main.go 파일을 열어서 다음과 같이 작성해보자. (저장까지 잘 하자)

package main

import "fmt"

func main() {
	fmt.Println("Hello, Go")
}

파일 실행

그런 다음, 터미널에서 다음과 같이 명령어를 실행하면 main.go 파일을 실행할 수 있다. (터미널 실행하는 경로에 main.go파일이 있어야 한다.)

go run main.go

main.go 파일을 실행하면 다음과 같이 출력한다.

Hello, Go
